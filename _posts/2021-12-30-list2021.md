> “每代人都必须发现自己的使命，并在相对的暧昧中实现或者背叛它。”

1. **海猫鸣泣之时** <br/> 这一整年里反复回顾的项目，到现在才更能客观分析一些，可以说既是故事的魔法也是叙事的魔法。首尾相连的完整意象符号体系，层层拆解之后得到的答案——安田其人，她/他所站立的苦难交叉点，所导向的作案动机，以人的主体和社会性为核心的事件因果——我是非常满意的，不过那主要是我个人对它的解读……作者本人的解读则是非常微妙，龙骑本质上是共情极高的温和保守派。对于安田本人的塑造方式我至今没有想通……无数的侧写，不知真相的脸和性格，却能衬托出非常鲜明的形象，写出了这样一个几乎没有在其他文艺作品里见过同类，但又的确很真实典型的角色。其他角色的性格和痛苦也都展现得非常好，有很多一针见血的社会素材，我个人还很喜欢绘羽/夏娃。距离拉远了之后也会意识到这种浓度的苦难其实是人为的极端情况，全都合情合理有代表性，但组合方式并不现实，是叙事的魔法让读者忽略了这一点，但确实也不影响文本意义的传达。

2. **MTMTE/LL** <br/> 大半年过去的最终感想还是……仍然很喜欢威震天（。）感谢这部作品贡献了承载着这种主题的这样一个角色，除了确实很符合我的审美以外，对他的好感也绝对有以这个角色来寄托自身对他所象征的那段历史那些概念的一部分思考的因素在内，历史的见证无数的尘埃，自我们中去又回到了何处……以及属于他个人的，被历史和大众推动着的生命轨迹，写下的理论，响应了与没能响应的祈祷，堕落的无回路，“你所做的一切都没有意义”，“为什么我仍然觉得我在浪费生命”，从轻发落，全都浪费了。Without love there is no meaning. 整体上也是始终很有诚意的作品，熟练的信息量和技法，十分真诚的道别，总之瑕不掩瑜，算是通俗作品里相当不错的了。

3. **回转企鹅罐** <br/> 很喜欢！个人觉得比少革好看，如果不考虑百合的因素的话。在人际关系等方面都相当激进——没有血缘关系的兄妹、从事过激派政治活动的父母，诸如此类，有很多、非常多很有意思和意义的主题元素，看过的几原作品里最复杂的一部。贯穿全作的符号也做得非常巧妙而动人，亲爱的未来，玻璃碎片，火焰，轮转的地铁标记，在手心里消散的一半的苹果。“谢谢你选择了我。” 

4. **百合熊岚** <br/> 很久以来看过最好的小美人鱼iteration，没有特别喜欢，不过倾向于把它视作企鹅罐的一种延续和解答，以及和少革之间的对称解构。镜子的意象太经典了。

5. **心灵奇旅** <br/> ……就挺有问题的，很好理解但我不是它的目标观众。

6. **罗小黑战记** <br/> 大概是普通的精致保守可爱电影……此外没什么特别高的评价。个人原因对无限比较无感（。小黑风息那段当时还挺喜欢的。

7. **极乐迪斯科最终剪辑版** <br/> “保持警惕，我爱你。”<br/> ——在所有方面都是满分，百分之两百的才能和诚意。ZA/UM我爱你们……

8. **狼行者** <br/> 挺失望的……觉得不好看，可能就是不该因为预告片有趣而对它产生不切实际的期待。有很多非常微妙的保守细节和悬置的议题，没法细想，但确实也没必要计较所以算了。很不喜欢主角和父母们像一家四口一样驾车远去的落幕方式……而且都什么年代了，怎么还让克伦威尔出来当万能反派啊。

9. **海洋之歌** <br/> 美术很漂亮，可能因为水彩的用色更明亮清澈丰富，我觉得比狼行者更漂亮。剧情没什么特别的，随便看看就行了。他们家可能就是叙事能力一般，没什么技巧性。

10. **A Bird Story** <br/> 寻找天堂前传，本质上是有互动元素的电影式小故事。借此回忆了一下去月球系列的风格，对我一个在玩rpg时经常找不到路的人来说这个系列都玩得很舒服流畅，设计显然用心了，对记忆的呈现方式细节也很巧妙。影子工厂什么时候出……

11. **恒水中学连环虐杀** <br/> 最喜欢开头，一些真正动人的天才的句子和设计，非常敏锐，充满关切。最后故事的谜底大概也是作者自己的挣扎，能看见这一点还蛮有意思的。

12. **不，我是乌克兰人** <br/> 只发布到第二章的未完结作品，非常神经病我相当喜欢，尤其喜欢现代线。现代线里的基辅几乎让人想起瑞瓦肖：被撕裂的、摇摆的城市，氛围里倾泻的无意义狂热与麻木。露骨到能直接看出作者借故事输出思想的打算，对我来说是加分项。总之拜托快点更新……

13. **不死鸟** <br/> 第一章写得太好了，而结尾也的确不行。意外的是花了整整三章认真讨论如何搞学生运动乃至于革命，普通人如何思考和参与，无法过审的程度远大于我想象……这么看他的作品其实一直都是很造反的。

14. **奇异人生** <br/> 我人生中浪费的16个小时。第一次知道游戏可以玩得这么痛苦，平心而论ep5最后还是有相当动人的地方，前面也有一些不错的细节，但整体还是不行。只想说讲故事的人的责任之一是让观众看见角色自己没能意识到的东西，奇异人生在这一点上是相当失败的。

15. **天气之子** <br/> 很有趣的片子……还是有技法拙劣转折生硬的地方，故事讲得不算最好，但主题和思路真的非常有意思。大概算是反对了现代工业社会的秩序常理，想要探寻一种更朴素自然的人际关系。喜欢那种理所应当的叛逆，尤其是没有按照“不给他人添麻烦”的原则安排剧情这一点。还有很多年后东京真的被大雨淹没，从东京搬走的成年人们却对主角的故事毫不在意，不觉得是他们改变了世界。

16. **姜子牙** <br/> 没啥感想，算是打卡了。至少比哪吒好。

17. **Luca** <br/> 看完直接忘了（

18. **自杀小队2** <br/> 喜欢这部！可以说是一种寻光号……快节奏黑色幽默键政血浆片的经典设计，键政成分比较可惜但以大IP商业片的标准而言已经很难得，毕竟好歹让美国政府当了反派，开头杀人竞赛那里的惊悚恐怖黑色幽默也很不错。以及哈莉美得我神魂颠倒。

19. **少女革命** <br/> 线索清晰主题明确，整体还是不错的，要点都到位了（拔剑角色的互换、当年见到的永恒等等）。大概少革就是把我想到的都写出来了，这已经是一种惊喜，而企鹅罐则是直接写出了我本来不敢想——或者说没有敢在这类大众向通俗作品里期待的。反复拉锯的部分是真的有点太多了，不过设计本身挺漂亮的。

20. **皿三昧** <br/> 就……不太行，无聊平庸还很中产小市民伦理，爱和欲望的主题仍然是固有线索，但企鹅罐和百合熊都做到了的事它直接倒退回解放前，纠结血缘的部分还不如少革。

21. **EVA终** <br/> 看完不记得自己看了个啥……单纯就是很难看以及需要光敏性癫痫预警。

22. **降世神通最后的气宗** <br/> 知名不具博主说它是欧美版钢炼这可能说得挺对的（。

23. **降世神通科拉传奇** <br/> 商业中间派liberal大作，中间派liberal浓度高达百分之八百，反派全是加起来都能凑一桌centricide的魔怔极端分子，写出来效果sb得仿佛自挂，看得我刷屏暴怒在各个群里流窜复读了大概八百遍我恨它，看完几个月后仍然觉得想要一个没有看过它的脑子。 <br/> <u>B1</u> <br/>故事的开场是如今进入了蒸汽时代，本作女主角Korra从乡下来到了由前代主角团在帝国侵略获取的土地上建造的共和国都城，“这个世界的和平与平衡的中心”（这不是自相矛盾么？），这座城市里有无家可归的流浪汉，男主角和他的兄弟则是在贫困中相依为命，一个去工厂做工一个卖艺；由于拥有驾驭某种元素的能力的御术师是天生的少数，所以也有黑帮打手用御术压迫普通人收保护费，普通人毫无还手之力。于是就有平权政治运动，翻译过来大约是叫平等教，是一个致力于铲除御术师的神秘过激邪教，整个动画把它写得非常过激，既不合理又阴谋论（不过后来发现他们只是剥夺别人的能力，基本上并不会杀人），而主角作为降世神通，也就是御术师的巅峰，在众星捧月的环境下长大，就自然而然地本能觉得它很不合理，当场与人激辩，表示“没有人压迫你们，是你们在压迫你们自己”。 <br/> ……一言以蔽之，就是为了能有东西可写，而加入了工业时代的设定，由此暴露出了一堆前作所没有的问题。整个故事的逻辑是非常分裂的，除了像fenlander所指出的“equal rights movement went too far”的伪命题以外，事实上哪怕单以故事的逻辑来看，在共和国城这一虚构的工业城市内部，就已经展现出了众多切实存在的不平等，而编剧却只能通过把反派的行为本身尽可能简化脸谱化，来强调他行为的荒诞不合理，让观众去支持主角。 <br/> 另外就是三角恋和体育竞赛的部分也很难看，主角团的人设全都健全现充到了不可理喻的地步，Asami和Mako那段写得宛如自挂，这里懒得复述了。……然而相对而言其实是最好玩的一季了，至少我骂它的时候心情最轻松愉快……反派的部分也是我比较喜欢的。 <br/> <u>B2</u> <br/> 纯粹的痛苦，难看得我几次三番看不下去，除了漫长的折磨以外就几乎没有任何记忆了，最后的超自然大战无语得我（。 <br/> <u>B3-4</u> <br/>让我的愤怒达到了新的巅峰的部分。 <br/> 剧情里讲的故事大概是这样的：超自然无政府主义小队打算让世界陷入原始的混乱来推行真正的自由，搞事时顺便随手谋杀了路过的国家的昏庸暴君。人民一开始普天同庆奔走相告，然而这个国家随后就陷入了民不聊生的混乱，良民被悍匪抢劫，此时世界的“平衡与和平的中心”的政府和议员对此什么都没做，疯狂踢皮球只打算扶持已故君主的傲慢废物公子哥儿子合法继位，然后就出现了临危受命统一国家带来稳定和物资和基础设施和清洗异己的强迫劳动营的集权大独裁者，主角团所做的事主要就是打完无政府小队打大独裁者，而那废物王子在最后也自动成长成了好人并主动放弃了君主制。 <br/> 也就是说，编剧认为无政府主义就是一小群人试图大力出奇迹让世界回归自然的混乱态的阴谋，这些人对于他们想要的自由没有任何符合人类社会性的阐释，行为也只是一味破坏；觉得人民群众毫无主动性，对生存环境的变化只会一味顺从，摆脱无道的昏君要靠随机路过的御术师（就当这只是由于武力碾压好了；那第一季怎么还敢说御术师对普通人没有任何压迫和不平等的事实？），而且就算失去了君主也不会开始自发组织；在这个前提下，即使全作唯一一次大快人心地铲除了苛政暴君的是这个无政府小队，即使在生民涂炭的时候唯一改善了现况的是后来的大独裁者，他们也仍然是大部分行为都极其脸谱化的反派，而什么都没做的共和国政府，却就算有种种缺点也仍然和主角团站在同一边，能够通过自省而得到改良；大独裁者那段我都不知道怎么说，完全是tankie负面刻板印象的集大成者，通过她这些纯粹负面的特质而合理化了主角团和共和国政府的一切行为…… <br/> 另外感情线仍然莫名其妙令人发指，第一次知道我能如此反感一个作品的官配百合cp，完全没有讨好到我这个真实存在的女同性恋。毕竟不是谁都是暴力无脑超自然中产救世主和资产阶级现充健全千金大小姐是吧（这cp哪怕单纯从二次元配对角度来看也极其无聊）。 <br/> <u>总结</u> <br/> 科拉传奇的剧情流程本质上是这样的：降世神通自己什么都不做，被动地由反派来改变世界，之后她就顺势而为地继续利用反派留下的遗产，甚至不会摈弃它们。第一季由旁白提及在平等教之后就有了关注非超自然人权益的政策之类的；第二季的Unalaq打开了人类世界和灵界的通道，之后就没有再关闭它，城市里长满了树木，精灵回到人世；第三季的小队杀死了昏君，大快人心，然后这地方就无政府了三年；第四季的Kuvira留下的运输等基础设施和工业技术大概率也在被继续使用。就算与世无争、拒绝参与权力争斗的动机在角色主观上可以自圆其说，整个剧情的进程也仍然暴露出了这类作品在根本上的逻辑弱点：为什么被动地在灾难降临时修补世界、而在修补完之后也只是让人民过着与原先相同的日子就是正派的全部行为，主动寻求变革的却都必定会成为走上极端的反派？前者成立的隐形前提，必然是在肯定着原本的生活就已经是人们所最为渴望的了，因此只要顺其自然就好，而后者则是对于打破这一生活的恐惧的具象化；只是实际上原本的生活或许也没有那么好，也有很多这类岁月静好中产视角以外的人其实难以忍受这样的日子，而将反派的实际建树纳入改良素材的行为也恰恰暴露了这一点。 <br/> 退一步说，就算可以认为子供向动画只要讲故事就好，认为设定里出现的问题并非它的本意，我就暂且同意这一点好了，但你的故事本身不也讲得稀烂吗？瞎挪用自己不懂的设定不也仍然非常傲慢吗？就算整个片子的目标受众只是小孩——虽然主角在故事开头的年龄是17岁，在三年后的第四季里甚至已经与我同龄——就算只是为了启蒙级别的娱乐和教育意义，那你又教了什么，天生神力的超自然救世主用暴力解决自己所遇到的一切问题？ <br/> <u>♠︎</u> <br/> 也有被一些无心插柳的地方打动过。第一季最后的Noatak的往事，荒无人迹的雪原，朴素的平等观念和对弟弟的保护，反抗父亲，后来的兄弟重逢，两次私奔的邀请。Mako在工厂里御火发电的那一幕，第四季的Bolin奶奶所居住的那个贫民社区，Wan的故事最开头的盗火。P'li和Zaheer的互诉衷情，是你让我知道了自由，明天我们会让世界都看见真正的自由。Bolin和其他Kuvira追随者一起分发物资救济灾民。 <br/> 喜欢的角色是Noatak和Kuvira，Kuvira我的报应女神…… <br/> 因为这些素材的潜力所以也想过搞一个重写，选定红莲教当年绑架Korra成功的if线，在设定方面也改写一下降世神通本身的象征意义，把重点从真正的超自然力上转移。设想里大致上是贫民窟少年英雄主角，和千金大小姐初恋又被背叛而分道扬镳，目睹养大自己的前辈战友们被共和国警队屠杀后彻底精神崩溃，最终转投以实干和理想说服了她的新锐势力领袖，而后者的帝国最终变成了和当年的前辈们的梦想完全相反的钢铁机器并土崩瓦解，除了苦难什么也没能留下，中间派共和国城赢了两次。 <br/> ……也还想说，什么Noatak被爹控制利用还护着弟弟，P'li从小是奴隶被Zaheer所搭救，Kuvira由于太固执叛逆而遭父母抛弃寄人篱下，也实在都是很动人的角色弧光开端，编剧在写的时候知道自己写出了什么吗……

24. **Onward** <br/> 去看是因为奇幻桌游的元素设定，但内核实在是很中产白男，又或者也许传统的奇幻桌游就是很白男。皮克斯这几年的风格也真是一以贯之的烂，这是在干什么（。

25. **Loving Vincent** <br/> 很容易看出受众，是真挚的作品。

26. **幽灵公主** <br/> 宫崎骏作品一贯的内核主题，故事写法不如风之谷，但很喜欢幻姬，很好奇作者本人对她这条线究竟是什么感想和态度……

27. **Psycholonials** <br/> 带有超自然外星文化殖民和LGBTQ元素的混乱反社会无政府（自称）左翼精神病恐怖分子造反故事，Andrew Hussie你确实是神经病……大部分时间都看得很爽，这个失手打死警察干脆开始策划反政府武装起义，详写了通过网络诈骗积累本金的过程，然后一路走上恐怖统治独裁屠杀的不归路的神经病展开我显然非常喜欢（。）主角Zhen的设定和性格也是一种要素过多，看了感觉分外亲切……音乐也都很好听。

28. **犹大与黑弥赛亚** <br/> 片头曲出来的时候就已经很难过了，成立彩虹联盟那里真的太好了……还有很多很多动人的地方，最后弹孔的特写，就像极乐迪斯科里逃兵那段台词写的，那些最贴心最勇敢的人，在你面前被无情地屠杀，所有的希望和柔情都被毁掉。FBI这边的对比也写得很好，O‘Neal的心理也意思到位了，不过女性成员相比之下很缺乏刻画。

29. **寻龙传说** <br/> Namaari有让我得到一点快乐，但作品本身wtf，连传说生物的不同身体部位分化为不同民族那种神话外壳的政治叙事都完全没改直接当做理所应当用进去了。题外话一下，什么时候才能有通俗作品明白无法意识到是自己没能理解对方的主角很讨厌……

30. **Imposter Factory** <br/> 故事非常简单无趣，无法独立于前作而产生任何意义，纯粹是讨好粉丝的产物，而且甚至没有在花心思讨好，不如说是钻空子利用了粉丝的感情。这么粗制滥造的东西完全是借着IP的底子就才能直接拿出来卖。

31. **We Know the Devil** <br/> 不错的小故事，尤其是苹果结局（？）。最会心一笑的部分竟然是看到Venus和Neptune一起抱怨想去赫奇帕奇却被官方测试分进了斯莱特林，然后开始讨论唐克斯看起来有多queer……

32. **Arcane** <br/> 意料之外地优秀的通俗动画，也算是IP衍生作品的顶点之一了，来自制作组的真诚的labor of love……编剧水平是可以的，上下城之间的矛盾在开篇就点得很明确，隔桥相望的两座城市的对比，多年的压迫和苦难，而角色的主题也都是围绕着这一核心矛盾展开。 <br/> 非常喜欢金克斯，本作的绝对女主角，她的角色本来就算是我会很有好感的类型，而结局对这个角色和这类作品而言也无疑是大胆且正当的升华，弧光的圆满。希尔科称她为他的女儿，而她又何尝不是象征意义与事实上的，下城的女儿……前期吸引我的是Powder这个角色的细节，基本上能和早期法斯相互代换，小高潮是第三集最后的大爆炸，整个社会结构与因果连锁里那种极度痛苦的戏剧性。中间喜欢过的角色关系是希尔科那边三位的办公室dynamics，金克斯和艾克之间也还算不错，不过艾克的剧情本身由于和执法官组搭线而一言难尽地比较黑洞。然后就是再也无法得到幸福的，也不再回头的结局，here's to the new us，最后那一炮看得我在屏幕前哭…… <br/> 角色关系问题。本来对姐妹关系其实挺无语的，尤其是最后蔚为了让金克斯停止发疯而对后者喊的那些话，本质上是在反复trigger对方，唤起的都是极端痛苦的回忆，感觉进行到这一步算是一切未来都彻底破裂，可以正式分手了。对jilco也一度同样挺无语虽然原因不同，总之是大脑不知道如何处理，只好调侃两句瘦子版本的埃弗拉特克莱尔半路收养早期法斯这样。然而真正看完之后倒是……对两边都完全没有这个困扰了，因为算是盖棺定论，这两段关系其实都是成就她这个角色的核心支架。结局其实是她物理上杀了希尔科这个养父，但希尔科是真正爱她的，作为姐姐的蔚才更接近于爹，她杀死希尔科之后在心理上也跟姐姐彻底了断，才有了最后那坚定的一炮，是复仇也是战书，这里真正的弑父是发生在姐妹之间的，切断了精神上最后的脐带。也就是说，在最后的谜底揭晓时角色就已经完全互换了，保护她又让她自由的是曾经也想利用她的希尔科，当年相依为命的姐姐到现在反而是为了自己而反复伤害她的，可以信赖的关系对象完全对调了一轮。是非常奇妙的设计了…… <br/> 那一炮也无疑的确是，把上城带来的暴力和苦难尽数归还给上城，是扰乱了原本的进程和轨迹的，推向历史的，在改良维稳的前夕点燃的炮火。 <br/> 另外上城这边杰斯线也写得很好，我也挺喜欢的。结局的多线抒情其实也有非常微妙的地方，但做得还是很漂亮，其余的就看编剧的实际立场和接下来的走向了。还有就是希尔科是真的跟埃弗拉特挺像的，能对上的有谋杀政敌上位当贫民窟黑帮老大，用微光控制范德尔曾经的交际圈和在工人饮食里加酒，依靠贩毒发家，整个生意的运转方式之类的，可能这个类型确实就是这样，可以加入一下素材库。

33. **魔法满屋** <br/> LMM的写歌水平还是可以的……剧本大概就是磷叶石虽然什么都做不了，但是通过给所有人话疗，解开了大家的心结而被接纳进了集体。非常阴间，所有角色都在焦虑自己哪天没用了怎么办然后得出所以我要继续有用的结论，而结局就是鬼片的巅峰，有用的方式是提供情绪价值所以要继续有用就要继续给所有人当一辈子树洞，直接绑定了，宛如看到什么在万众欢呼下和家暴男步入婚姻殿堂……总而言之就是让磷叶石通过给集体当一辈子桃果的方式达成he。 <br/> 零碎的离谱地方吐槽：让主角来安慰祖母——正常情况不该是反过来吗；舅舅的角色——这都快要成为性转版的阁楼疯女人了；开头主角绝望地对祖母唱睁开你的眼睛看看我吧，在结尾得到的回应竟然是祖母对主角唱睁开你的眼睛看看你自己，连一句我看见你了的正面回答都没有；最后大家鼓掌认可接纳主角——你们不是至少该道歉吗？
